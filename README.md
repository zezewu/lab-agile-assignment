# HDMart Handmade E-commerce Agile Project
This repository contains the agile planning and development artifacts for HDMart, an online marketplace for handmade products. The project follows Scrum methodology and uses GitHub + ZenHub for project management.

## Project Overview
HDMart is a startup building a simple, mobile-friendly e-commerce platform for handmade goods. The goal is to deliver a Minimum Viable Product (MVP) in 6 sprints (3 months) with continuous iteration based on customer feedback.

### Core Features (MVP)
- Product search and browsing
- Shopping cart management
- Secure checkout and payment
- Seller product upload and management
- Multi-device responsiveness

## Tech Stack
- Project Management: GitHub Issues, ZenHub
- Deployment: GitHub Pages
- Documentation: Markdown
